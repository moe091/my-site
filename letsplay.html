Let's Play - 100 Card, Dice, Domino, and Dart Games!

Tech:
Android, Java, HTML, CSS

Let's Play is an Android app I created for a freelance client. My client had written a book about different card, dice, domino, and dart games containing rules, how to play, histories of the games, and other information, and he wanted an App to sell along with the book. He gave me the content of the book, formatted in html and css, and asked me to create an Android that displayed the books content organized inside of an easy-to-use UI and also contained a dynamic scoreboard that would work for any combination of game-type, number of players, and number of rounds.

The project actually consisted of several apps, mainly the '100 Games' version, but also separate versions for just cards, darts, dominos, or dice games. All of the apps were very similar to each other though and creating the different versions didn't add any new technical challenges to the project. In the section below you'll find links to the main '100 Games' app as well as the '25 Card Games' app on the app store.

Instead of tediously copying every single page of the books content into Android Activities and layouts, I just created the basic UI flow and menu's and used a modified version of Android's WebView component to load the pages he gave me. The default WebView implementation took a second to load after the screen for that section of the book loaded, so I modified the code to pre-load the content for the pages to be displayed based on the users location in the menu so that when the user clicked on a game, the preloaded WebView could be displayed instantly. The preloading didn't have any noticeable performance side-effects and worked well across all Android devices tested.
